Функциональность:
* единая версия для телефонов и планшетов
* поддержка разрешений вплоть до xxhdpi
* поддержка разных версий Android начиная с 4.0.3 (API v14)
* пауза при входящем и исходящем звонках
* управление воспроизведением с гарнитуры (не для всех устройств)
* различные режимы интерполяции с оптимизацией по скорости работы и по качеству звука
* кеширование треков из плейлиста на SD карте
* виджет для главного экрана
* возможность отправлять модули и делиться ссылками на модули с http://zxtunes.com, http://www.zxart.ee/rus/muzyka и http://modarchive.org
* возможность установить модуль на входящий звонок
* поиск и воспроизведение треков из архивов

Используемые разрешения и их назначение:
* доступ к интернету/статус сетевого соединения для доступа к сетевым источникам данных
* чтение/изменение содержимого SD-карты для доступа к локальным источникам данных и экспорта списков
* предотвращение перехода в режим ожидания для ускорения работы сервиса сканирования
* изменение системных настроек для установки модуля на звонок
